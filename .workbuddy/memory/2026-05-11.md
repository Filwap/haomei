# 2026-05-11 工作日志

## haomei 项目 - Token 认证修复 (commit f8534dc)

**问题**：照片墙管理后台删除照片时出现 401 Unauthorized 错误
**根因**：浏览器 localStorage 中残留旧格式 JWT token（`eyJ...`），后端已迁移到新格式 token（`timestamp.signature`，含 `.` 分隔符）
**本次修复内容**：
- 将旧 token 清除逻辑封装为 `cleanOldToken()` 函数，检测条件增加 `!authToken.includes('.')` 覆盖更多情况
- 新增 `ensureValidToken()` 防御函数，在每次 API 请求前验证 token 格式有效性
- 无效 token 时自动清除 localStorage 并跳转登录页
- 彻底解决因 Cloudflare CDN 缓存导致旧版 JS 中 JWT token 仍被发送的问题

**用户需要做的操作**：
1. 等待 Cloudflare Pages 部署完成（约1-2分钟）
2. **Ctrl+Shift+R** 强制刷新浏览器清缓存
3. 打开开发者工具 Application > Local Storage → 手动删除 `hm_admin_token`
4. 重新登录
